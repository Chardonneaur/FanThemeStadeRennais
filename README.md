# FanThemeStadeRennais

Unofficial fan-made Matomo theme inspired by Stade Rennais FC — Rouge et Noir.

> **Warning**
>
> This plugin is experimental and was coded using [Claude Code](https://claude.ai).
> It is provided without any warranty regarding quality, stability, or performance.
> This is a community project and is not officially supported by Matomo.
>
> **This theme is not affiliated with, endorsed by, or associated with Stade Rennais FC.**

## Description

Transform your Matomo analytics dashboard into Roazhon Park colours.
The theme applies the club's iconic **Rouge (#E30613)** and **Noir (#1A1A1A)** throughout
the Matomo interface:

- **Navigation header** — Noir with white text
- **Buttons & interactive elements** — Rouge Rennais
- **Links & active menu items** — Rouge accents
- **Graph series** — Alternating rouge and noir palette
- **Visitor map** — Red gradient choropleth
- **Scrollbar** — Subtle rouge accent
- **Background** — Faint diagonal kit-stripe pattern

## Requirements

- Matomo >= 5.0.0
- PHP >= 8.1

## Installation

### Manual Installation
1. Download the latest release from GitHub
2. Extract to your `matomo/plugins/` directory
3. Go to **Administration > Appearance** and activate **FanThemeStadeRennais**

### Via Marketplace
1. Go to **Administration > Marketplace**
2. Search for **FanThemeStadeRennais**
3. Click Install, then activate in Appearance settings
