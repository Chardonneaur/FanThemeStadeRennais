# FAQ

**Is this plugin official?**
No. This is an independent fan project. It is not affiliated with or endorsed
by Stade Rennais FC.

**Does it affect my data?**
Not at all. The theme is purely cosmetic — it only changes colours and styles.

**Does it work with Matomo 4.x?**
The theme targets Matomo 5.x. It may work on 4.x but has not been tested.

**Can I combine it with other plugins?**
Yes. It's a standard Matomo theme and is compatible with all plugins.

**The colours look wrong after an update.**
Clear the asset cache: go to **Administration → Diagnostics → Delete Cache** or run
`php console cache:clear` from the Matomo root.
