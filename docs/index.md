# FanThemeStadeRennais Documentation

## Overview

FanThemeStadeRennais applies the Stade Rennais FC colour palette to the Matomo
analytics interface. It is a purely visual plugin — no data collection, no tracking,
no database changes.

## Colour Palette

| Role | Colour | Hex |
|------|--------|-----|
| Brand / Buttons | Rouge Rennais | `#E30613` |
| Header Background | Noir | `#1A1A1A` |
| Header Text | Blanc | `#FFFFFF` |
| Links | Rouge foncé | `#C00010` |
| Active Menu | Rouge Rennais | `#E30613` |
| Page Background | Gris clair | `#F5F5F5` |

## Activation

Go to **Administration → Appearance** and select **FanThemeStadeRennais** from
the theme list, then click **Save**.

## Deactivation

Select any other theme (e.g. the default Morpheus) in the same Appearance panel.
